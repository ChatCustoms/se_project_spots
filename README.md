# Spots website

This is the third project of the Software Engineering program at TripleTen. It was created using HTML and CSS, based on the design brief from Figma.

## Project features

- Semantic HTML5
- Flexbox
- Positioning
- Flat BEM file structure
- A custom form
- CSS animation and transform
- Responsive design
